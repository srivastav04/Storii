# 📸 Storii (Previously Postly)

Storii is a full-stack social platform for sharing stories with **images & videos**, built with a modern TypeScript-first stack. It focuses on scalability, clean UX, and a responsive, delightful UI.

## 🚀 Tech Stack

### 🧩 Frontend
- **Next.js (App Router)** — SSR/ISR + client components where it counts  
- **TypeScript** — end-to-end types  
- **Hero UI** — sleek, accessible UI primitives  
- **Tailwind CSS** — rapid, consistent styling  
- **React Query** (optional) — easy caching & optimistic UX

### ⚙️ Backend
- **NestJS** — modular, testable API  
- **Prisma** — type-safe ORM  
- **MongoDB**  — Database  
- **ImageKit** — media uploads, transformations, and CDN

---

## ⚡ Key Features
- ✏️ **Set & edit profiles** (display name, bio, avatar, banner)
- 🔎 **Search users** with partial match (name/email)
- 🎥 **Upload videos** via ImageKit with signed authentication
- 📱 **Fully responsive UI** across breakpoints
- 🔐  **Authentication** using Clerk
